# Image processing

## 🤝 Contributing


Contributions are welcome.

Here is an example of adding a new feature to the code:

```
git switch -c feature/your-awesome-feature
# ... make changes ... git add .; git commit -m "Useful commit message"
git push --set-upstream origin feature/your-awesome-feature
xdg-open https://github.com/dcintlab/image-processing/pull/new/feature/your-awesome-feature
# ... merge the branch to the master, make sure that the pipeline passes, delete the branch.
git switch main
git pull
git branch -d feature/your-awesome-feature
git branch -d origin/feature/your-awesome-feature --remote
```
